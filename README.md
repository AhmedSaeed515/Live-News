# 📰 Live News App

> A modern, real-time news application delivering the latest headlines with a clean and responsive user experience.

---

## 📌 Overview

**Live News** is a mobile application built to provide users with up-to-date news from multiple sources. The app focuses on performance, usability, and a clean UI to ensure a seamless reading experience.

---

## ✨ Features

* 🔴 Real-time news updates
* 🔍 Search functionality for articles
* 🗂️ Category-based news browsing (e.g., Sports, Technology, Business)
* 📱 Responsive and intuitive UI
* ⚡ Fast data loading with API integration

---

## 🛠️ Tech Stack

* **Frontend:** Flutter
* **Language:** Dart
* **API:** RESTful News API
* **State Management:** (Provider / Bloc / setState — update based on what you used)

---


## 🚀 Getting Started

### Prerequisites

* Flutter SDK installed
* Android Studio / VS Code
* Emulator or physical device

---

### Installation

```bash
git clone https://github.com/AhmedSaeed515/Live-News.git
cd Live-News
flutter pub get
flutter run
```
